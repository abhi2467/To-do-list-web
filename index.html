Help & Guide

Welcome to To Do List App

This guide will help you get started with the To Do List application. Follow these simple steps to manage your tasks efficiently.

1 Adding Tasks

To add a new task:

  1. Type your task in the input field at the top of the page
  2. Select a priority level (Low, Medium, or High)
  3. Click the "Add Task" button or press Enter
2 Managing Tasks

Once tasks are added, you can:

  1. Mark tasks as complete by clicking the checkbox
  2. Edit tasks using the edit button
  3. Delete tasks using the delete button
  4. Filter tasks using the filter buttons
3 File Management

Your tasks are automatically saved. You can also:

  1. Export tasks to a JSON file for backup
  2. Import tasks from a JSON file
  3. View your tasks in JSON format
  4. Clear all tasks when needed
Pro Tip: Use priority levels to categorize your tasks. High priority tasks have red borders, medium have blue, and low have green.

Frequently Asked Questions

Find answers to common questions about the To Do List app:

Where are my tasks stored?

Your tasks are stored locally in your browser's localStorage. This means they are saved on your computer and will persist between browser sessions, but they won't sync across devices unless you export and import them.

How do I backup my tasks?

You can backup your tasks by clicking the "Export to JSON" button in the File Management section. This will download a JSON file with all your tasks. You can later import this file to restore your tasks.

Can I use the app offline?

Yes! The To Do List app works completely offline. All the code runs locally in your browser, and tasks are stored locally. You don't need an internet connection to use the app after the initial page load.

How do I clear all my tasks?

Click the "Clear All Tasks" button in the File Management section. You'll be asked to confirm before all tasks are permanently deleted. Make sure to export a backup first if you want to keep a copy.

Can I change the priority of a task?

Yes! You can edit any task by clicking the edit button (pencil icon) next to the task. In the edit dialog, you can change the task text and update the priority level.

Is my data secure?

Your data is completely private and stored only on your device. Since the app runs locally and doesn't connect to any servers, your tasks are only accessible from your browser on this device.

How do I filter tasks?

Use the filter buttons above the task list. You can show all tasks, only pending tasks, only completed tasks, or only high priority tasks. The active filter is highlighted in blue.

Tips & Tricks

Maximize your productivity with these expert tips:

Productivity Tips
  • Prioritize Effectively: Use high priority for urgent tasks, medium for important but not urgent, and low for routine tasks
  • Break Down Tasks: Large projects can be broken into smaller, manageable tasks
  • Daily Review: Start each day by reviewing and prioritizing your task list
  • Use Completion Tracking: Marking tasks complete provides a sense of accomplishment
Organization Strategies
  • Consistent Naming: Use clear, descriptive names for your tasks
  • Regular Backups: Export your tasks weekly to prevent data loss
  • Filter Views: Use different filters for different working modes
  • Color Coding: Use the visual priority indicators to quickly identify important tasks
Advanced Features
  • Quick Add: Press Enter after typing to quickly add tasks without reaching for the mouse
  • Bulk Operations: Use the clear all feature when starting fresh projects
  • Data Portability: Export your tasks to share with team members or use in other apps
  • Keyboard Navigation: Learn the keyboard shortcuts for faster task management
Expert Tip: Create a "Daily" task list by exporting your current tasks, clearing the list, and adding just today's tasks. At the end of the day, import your master list back and mark completed items.
